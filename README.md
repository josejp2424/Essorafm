# Essorafm
Modular GTK3 file manager for Essora Linux and any non-systemd distro. Tabbed browsing, XDG trash, rsync-backed copy, side preview panel for images/PDF/TXT/EPUB, duplicate scanner, network bookmarks (SMB/FTP/SFTP), pkexec/gksu privilege escalation, 12-language UI. Pure Python 3 + GTK3, OpenRC-friendly.
